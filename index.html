How to do Real Scans (Legally) with Nmap

Run these commands on your own lab machines or with written permission only: 

nmap -sV target-ip          # Service/version detection
nmap -A target-ip           # Aggressive scan (OS, version, scripts)
nmap -p- target-ip          # Scan all 65535 ports
nmap --script vuln target   # Run vuln scripts (authorized targets only)
